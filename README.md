# Student Record System

A console-based application built in C++ that allows user to manage student information efficiently. This project demonstrates core programming concepts including Object-Oriented Programming (OOP), arrays, and advanced control flow structures.

## 🚀 Features
* **Add Student Records:** Capture and store student details (ID, Name, Grades, etc.).
* **View Records:** Display a comprehensive list of all enrolled students.
* **Search & Filter:** Instantly look up a specific student using their unique ID.
* **Dynamic Menus:** A smooth, user-friendly command-line interface.

## 🛠️ Core Concepts Demonstrated
This project was built to practice and showcase fundamental software development concepts:
* **Classes & Objects:** Used to model a `Student` and encapsulate their attributes and behaviors.
* **Arrays:** Employed to store and manage multiple student records sequentially.
* **Switch Statements:** Handles the main menu navigation cleanly.
* **If/Else Statements:** Validates user input and handles conditional logic (e.g., checking if a record exists).
* **For Loops:** Iterates through arrays to display or search for specific student records.
* **While Loops:** Drives the main application loop, keeping the program running until the user chooses to exit.
